# Brandon's Dotfiles
This repo contains my personal dotfiles, which will probably change and update constantly. Feel free to fork and use anything you find here. Some of the settings here tend to be more towards my Linux configs and will have to try and clean some of this up when I add MacOS configs.

**Warning**: Be sure to backup your own dotfiles before replacing them. 

## Neovim / NVChad / LunarVim
I am currently jumping between [NVChad](https://nvchad.com/) and [LunarVim](https://www.lunarvim.org/). I am using MVChad as a jumping off point along with LunarVim while I slowly being to build my own custom config.

## [Hyprland](https://hyprland.org/) (Nvidia)
Hyprland config based off of [JaKooLit](https://github.comhttps://github.com/SolDoesTech/JaKooLit) and [SolDoesTech](https://github.com/SolDoesTech)

### Components:

- Main Component: [`Hyprland`](https://github.com/hyprwm/Hyprland)
 
- Terminal: [`Kitty`](https://sw.kovidgoyal.net/kitty/)

- Shell: [`Zsh`](https://ohmyz.sh/)
 
- Status bar: [`Waybar`](https://github.com/Alexays/Waybar)
 
- Menu : [`Wofi`](https://hg.sr.ht/~scoopta/wofi)

- FIle Manager: [`Thunar`](https://docs.xfce.org/xfce/thunar/start)
